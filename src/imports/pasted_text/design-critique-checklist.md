# 🔍 LIKELY ISSUES (and how to fix them)

These are VERY common in Figma AI outputs — check if your design has them:

---

## ❌ 1. Too Much Focus on Sleep (Wrong Priority)

### Problem:

If your UI looks like:

* sleep stats
* calming visuals
* tracking

👉 then it's drifting into a **generic sleep app**

### ✅ Fix:

Make sure:

* Home screen = **timeline first**
* Sleep = **just one protected block**

👉 Sleep should feel like:

> “result of good scheduling”
> not
> “main feature”

---

## ❌ 2. Schedule Looks Static (BIG ISSUE)

### Problem:

If your schedule is just:

* a list
* or fixed timeline

👉 then your **USP is missing**

### ✅ Fix:

Add visible intelligence:

* Labels like:

  * “Auto-adjusted”
  * “Optimized for sleep”
* Moving blocks animation (even implied in UI)
* Highlight:

  * “Recently changed”

👉 It should feel **alive**

---

## ❌ 3. No “Execution Pressure”

### Problem:

Most designs only show plan, not action.

### ✅ Fix:

Add:

* ⏳ “Start now” button on current task
* 🔔 Status:

  * “You’re behind”
* ⏱ Countdown for current task

👉 Your app should feel like a **coach**, not planner

---

## ❌ 4. Weak Add Task Flow

### Problem:

If adding a task is too simple (just name + time)

👉 then your AI logic breaks

### ✅ Fix:

Ensure UI includes:

* Task type (exam / homework / etc.)
* Deadline
* Estimated time
* Priority

👉 This is CRITICAL for your backend later

---

## ❌ 5. No Clear Sleep Protection Indicator

### Problem:

User doesn’t know if they’re “on track”

### ✅ Fix:

Add at top:

* ✅ Sleep secured
* ⚠️ Sleep at risk

👉 This creates urgency + clarity

---

## ❌ 6. Wind Down Mode Not Visually Distinct

### Problem:

If it looks similar to main UI → boring

### ✅ Fix:

Make it:

* Dark gradient
* Soft glow
* Slower feel

👉 Should feel like entering a different “mode”

---

## ❌ 7. Too Cluttered Dashboard

### Problem:

Figma AI often adds:

* too many cards
* too much info

### ✅ Fix:

Your dashboard should ONLY have:

* Timeline
* Sleep indicator
* 3 buttons

👉 That’s it.

---

## ❌ 8. No Feedback When Schedule Changes (CRITICAL)

### Problem:

User doesn’t understand what happened

### ✅ Fix:

Add micro UI:

* “Schedule updated”
* “Math moved to 8:30 pm”

👉 This builds trust in your AI
