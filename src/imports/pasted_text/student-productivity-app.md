Design a mobile application UI for a smart student productivity and sleep improvement app.

The app’s core purpose is to help students manage their daily schedule in a realistic and adaptive way so they can consistently sleep on time. The focus is on dynamic scheduling, execution, and reducing procrastination. Sleep-related features are secondary and supportive.

Use a dual design system:

1. Productivity Mode: minimal, clean, structured layout with neutral colors (white, grey, light blue), grid-based design, highly readable
2. Sleep Mode: dark theme with soft gradients (purple, navy, blue), calming, immersive, slightly animated

The UI should feel intelligent, adaptive, and action-oriented — not just analytical.

Design the following screens in order:

1. Splash Screen

* App logo and tagline: “Plan your day. Protect your sleep.”

2. Onboarding Flow (4–6 screens)

* Goal selection (better sleep, reduce stress, manage time)
* Sleep habits input (current sleep time, ideal sleep time, wake time)
* Daily schedule input (school timings, classes, extracurriculars)
* Biggest struggles (phone addiction, procrastination, workload, exam stress)
* Energy pattern (morning/night person, peak focus hours)

3. Home Dashboard (MOST IMPORTANT SCREEN)

* Top section:

  * Greeting
  * Sleep countdown (e.g. “Sleep in 2h 15m”)
  * Sleep indicator (“Sleep secured” or “Sleep at risk”)
* Middle section:

  * Timeline-based daily schedule with time blocks
  * Tasks, breaks, and sleep block clearly visible
  * Sleep block visually distinct and locked
* Bottom section:

  * Primary buttons: “Plan My Day”, “Start Focus”, “Wind Down”

4. Plan My Day Flow

* Task input screen:

  * Task name
  * Task type (homework, exam, assignment, class)
  * Deadline
  * Estimated time
  * Priority
* Preferences screen:

  * Sleep goal
  * Energy level
  * Work intensity
* AI generation screen:

  * Animation with text like “Building your schedule…” and “Optimizing for sleep…”
* Generated schedule screen:

  * Clean timeline layout
  * Color-coded tasks:

    * Red (urgent)
    * Yellow (normal)
    * Blue (flexible)
  * Includes breaks, buffer time, wind-down routine, and a locked sleep block
* Editable schedule screen:

  * Drag-and-drop task blocks

5. Dynamic Scheduling System (IMPORTANT)

* Show that the schedule updates automatically when:

  * New tasks are added
  * Tasks are delayed or skipped
* Include UI elements:

  * “Rebuilding your schedule…” animation
  * Smart update messages:

    * “Schedule updated to keep your sleep on track”
    * “Task moved due to priority”
  * Highlight changed tasks in the timeline

6. Focus Mode Screen

* Ultra minimal UI
* Large timer
* Current task
* Progress indicator
* No distractions
* Show app blocking status

7. Wind Down Mode (Visually Rich)

* Dark gradient UI with soft animations
* Options:

  * Music player
  * Breathing exercise
  * Bedtime story
* Calm, immersive interface

8. Progress Screen

* Schedule adherence percentage
* Sleep consistency
* Streak tracking
* Simple visual graphs

9. Settings Screen

* Sleep goal settings
* Notification settings
* App blocking controls
* Personalization options

Design requirements:

* Modern, student-friendly interface
* Minimal but smart in productivity screens
* Visually engaging in sleep mode
* Smooth transitions and micro-interactions
* Clear hierarchy and readability
* Avoid clutter and avoid overly complex dashboards

The scheduling system should feel dynamic, adaptive, and intelligent, always prioritizing sleep while adjusting tasks around it.
