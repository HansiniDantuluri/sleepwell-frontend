Redesign and enhance a mobile application UI for a smart student productivity and sleep improvement app. The app is already functionally complete, so focus ONLY on improving visual design, engagement, and user experience without changing core functionality.

Core principle:

* The scheduling system must remain minimal, clean, and highly readable
* Exciting visuals, animations, and gamification should be added to supporting features (focus mode, wind-down, progress, micro-interactions)

Target users:

* Students aged 13–18 (grades 8–12)
* The design should feel modern, slightly playful, motivating, and not boring

Use a dual design system:

1. Productivity Mode:

   * Minimal, structured, clean layout
   * Neutral tones with subtle gradients (white, grey, soft blue)
   * Clear hierarchy and spacing
2. Experience Mode (Focus + Sleep):

   * Rich gradients (purple, navy, blue)
   * Soft glow effects
   * Smooth animations and immersive visuals

Enhance the following screens:

1. Home Dashboard (Keep minimal but add personality)

* Add a smart dynamic header with messages like:
  “You’re on track today 🔥” or “Let’s protect your sleep tonight 🌙”
* Add a circular “Sleep Protection Indicator” ring:

  * Green (safe), orange (risk), red (danger)
* Add an “Energy Level Bar” showing focus levels throughout the day
* Keep the timeline clean but add:

  * Smooth transitions when tasks shift
  * Highlight “recently adjusted” tasks
* Add a “Current Focus Card”:

  * Shows current task, time remaining, and a “Start Focus Mode” button

2. Dynamic Schedule Feedback (Important)

* Show visual feedback when schedule updates:

  * Animated movement of task blocks
  * Highlight changed tasks
  * Notification banners:
    “Schedule updated to protect your sleep”
    “Task moved due to priority”
* Include a subtle “Rebuilding your schedule…” animation state

3. Focus Mode (Make it gamified and engaging)

* Add a large animated timer with a circular progress ring
* Include a “Focus XP” or progress system:

  * Points gained per session
* Add a “Focus Meter” that fills as the user stays on task
* Include a visual element like a growing plant or progress animation
* Add subtle gradient backgrounds and glowing effects
* Keep distractions minimal but visually engaging

4. Wind Down Mode (Highly immersive and aesthetic)

* Full dark gradient interface (deep blue, purple, black)
* Add animated backgrounds (stars, slow-moving gradients)
* Include:

  * Breathing animation (expanding/contracting circle)
  * Music player with sliders (rain, piano, white noise)
  * Bedtime story cards (swipeable UI)
* Add smooth transitions when entering sleep mode (screen gradually darkens)

5. Progress Screen (Make it rewarding and motivating)

* Add gamification elements:

  * Streak counters (fire icon or similar)
  * Achievement badges (e.g. “3 days on time sleep”)
* Show improvements:

  * “You’re sleeping 40 minutes earlier than last week”
* Use simple but visually appealing charts
* Add a sense of progression and growth

6. Micro-interactions across the app

* Task completion → checkmark animation with slight bounce
* Schedule updates → smooth shifting of blocks
* Notifications → modern slide-in banners (not basic alerts)
* Entering sleep mode → smooth fade and darkening
* Buttons → subtle hover and tap animations

7. AI Personality Layer

* Replace generic system messages with friendly, intelligent messages:

  * “I adjusted your schedule to keep your sleep on track”
  * “You’re running a bit behind — let’s catch up”
* Design UI elements that feel like a smart assistant guiding the user

8. Visual Style Enhancements

* Use soft gradients instead of flat colors
* Add depth with shadows and layered cards
* Use rounded corners and modern spacing
* Avoid clutter — maintain simplicity in core scheduling screens

The final design should feel:

* Intelligent and adaptive
* Slightly playful and motivating
* Clean in productivity areas
* Visually rich in focus and sleep experiences
* Designed specifically for students, not generic users
